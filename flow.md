First, borrower(person) borrows an item, next admin then lists the items borrowed using the system. An item has its own QR code based on the inventory tab. Admin scans this and the system shows the item and then admin inputs the name of the borrower, contact number, date borrowed, date to return borrowed, Remarks, location and remarks item under the item borrowed. After that, a confirmation modal will pop up displaying information about the borrowed item. In the borrowed items tab, this is what will be inside the table.

---------------------------------------------------------------------------------
 No. |  Item Name  | Qty | Borrower | Contact No. | Date Borrowed | Return Date | 
---------------------------------------------------------------------------------

And then when a row is clicked it will show a modal that will show the information for the row clicked. It will display the No., Name of the borrower, Contact No. of the borrower, location, Item borrowed, Quantity, Amount of the borrowed item/s, Date borrowed, Return Date, Remarks
