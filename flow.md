First. User will select the incoming staff and if they are going to turnover the inventory or the vehicle. After Selecting, proceed to the next page which will show a table of the items to be audited. The table will show this:


-----------------------------------
item | condition | Qty | Final Qty
-----------------------------------
item | dropdown (good, defective, missing, used, increased) | Qty | Final Qty

User can select multiple items in dropdown. For example they selected Good, they will then input the Qty of the items in Good condition. And then after, they click a plus sign under the selected dropdown to add another entry for the same item, then they selected bad and input the qty of the items in bad condition. If there are missing items, deduct it to the qty and show it in Final Qty, if the item is increased, increase the Finaly Qty
